# 試合登録・オーダー入力（Step 2）

## 概要

試合の登録とオーダー（打順・守備位置）入力機能を実装する。
トップ画面にチーム一覧に加えて試合一覧を表示し、試合詳細画面ではオーダー表を確認できる。

## DB制約メモ

- `games.status`: `scheduled` | `in_progress` | `finished`（デフォルト: `scheduled`）
- `lineups.team_side`: `home` | `visitor`（自チームがホームなら `home`、ビジターなら `visitor`）
- `lineups.batting_order`: 1〜9
- `lineups`: `player_id` か `player_name` のどちらかが NOT NULL 必須
- RLS: `games`, `lineups`, `pitching_records` はすべて `team_members` 経由の所属チェック

## 実装メモ

- `team_side` は `is_home` に応じて自チーム = `home`/`visitor`、相手 = 逆側を設定する
- 相手チームのオーダーは `player_id = null`、`player_name` に名前を設定（省略時は「相手選手N」）
- 先発投手の `pitching_records` は `inning_from = 1` で作成
- v_scoreboard ビューでスコアを取得（終了試合のスコア表示用）
