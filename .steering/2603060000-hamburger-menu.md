# モバイル時のナビゲーションをハンバーガーメニューにする

関連 issue: #2

## 概要

モバイル表示時、ヘッダーのナビゲーションをハンバーガーメニュー（スライドイン）に変更する。
デスクトップ/タブレットでは従来のヘッダーナビゲーションを維持。

## 背景

- 試合記録中（input画面）は画面領域を最大限確保したい
- メニュー項目が多い
- タブレットメインなのでハンバーガーの発見性は問題にならない

## 実装方針

- `app/(main)/layout.tsx` のヘッダーを修正
- モバイル幅（`md`未満）でハンバーガーアイコンを表示、navリンクを非表示
- shadcn/ui の Sheet コンポーネントでスライドインメニューを実装
- ハンバーガーメニュー内にナビリンク + 設定 + ログアウトを配置
- Client Component として `MobileNav` コンポーネントを分離（Sheet は state が必要）
