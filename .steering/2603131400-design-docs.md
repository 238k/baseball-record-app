# 設計ドキュメント・コード品質ルールの整備

関連 issue: #24

## 概要

暗黙的に使われているルールを明文化し、今後の開発品質を担保するためのドキュメント群を作成する。
CLAUDE.md にコード品質ルールを追加し、各ドキュメントで詳細を補完する構成。

- 各ドキュメントは「現状のパターン」と「目指すべきパターン」の両方を記載

- 日本語で記述（コードベースのコメントが日本語のため）
- 既存の良いパターンは積極的に引用してルール化
